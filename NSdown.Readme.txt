NSdown::Download	/URL url [/FILE path]
					[/RETRYCOUNT count] [/RETRYDELAY msec]
					[/CONNECTRETRIES count] [/CONNECTTIMEOUT msec] [/RECEIVETIMEOUT msec]
					[/PROXY server] [/PROXYUSER user] [/PROXYPASS pass]

/URL url					| HTTP or HTTPS
/FILE path					| Local path where the file is downloaded. If not specified, no content is downloaded, only headers.
/RETRYCOUNT count			| The number of InternetOpenUrl calls. Default is 5.
/RETRYDELAY msec			| Delay between two InternetOpenUrl calls. Default is 1000.
/CONNECTRETRIES count		| InternetSetOption( INTERNET_OPTION_CONNECT_RETRIES ). Works only for hosts with multiple IPs!
/CONNECTTIMEOUT msec		| InternetSetOption( INTERNET_OPTION_CONNECT_TIMEOUT )
/RECEIVETIMEOUT msec		| InternetSetOption( INTERNET_OPTION_RECEIVE_TIMEOUT )
/PROXY server				| Not implemented yet. The default proxy settings are used
/PROXYUSER user				| Not implemented yet. The default proxy settings are used
/PROXYPASS pass				| Not implemented yet. The default proxy settings are used

Examples:
NSdown::Download /URL "http://live.sysinternals.com/Files/SysinternalsSuite.zip" /FILE "$EXEDIR\SysinternalsSuite.zip" /RETRYCOUNT 5 /RETRYDELAY 5000
